## About TSUBUYAKI BOOK　『このWebアプリケーションの概要』
このアプリは、読書が好きな方がおすすめの本を紹介しあうWebアプリケーションになっています。

## TSUBUYAKI BOOKの機能一覧
- 投稿機能
- 投稿にコメント機能
- ログイン・ログアウトのような認証機能
- ページネーション機能
- いいね機能
- 投稿の削除機能
- 投稿の編集機能
